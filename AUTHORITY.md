# Authority Boundary

## Current authority

This repository grants **no production authority** and **no autonomous high-stakes authority**.

Authority is derived from verified evidence and an explicit gate decision. It is never derived from configuration, repository ownership, steward identity, test count, cryptographic formatting, or symbolic declaration alone.

## Prohibited uses

Until an applicable promotion gate is satisfied and recorded, repository artifacts must not be represented or used as:

- a production AI service;
- a clinical, diagnostic, therapeutic, legal, financial, or emergency decision system;
- an autonomous high-stakes actuator;
- a hardware-enforced safety mechanism without physical timing receipts;
- a formally verified system without machine-checked proof logs;
- an AGI safety certification or regulatory certification;
- a self-certifying evidence authority.

## Fail-closed rule
